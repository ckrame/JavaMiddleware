Generates XMLPullParser classes (Factory generates class name automatically).

XMLPullParser is an interface which defines parsing functionality provided in XMLPULL V1 API. Parsers can be both validating or non validating.

XMLPullParserFactory

will generate XMPPullParser implementing classes with names based on the parameters passed as arguments

The XmlSerializer interface is included as part of preparing a possible migration towards XmlPull 1.2 API
