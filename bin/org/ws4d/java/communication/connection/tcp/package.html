Platform independent server socket/connection and TCP connection framework.

Defines two interfaces Socket and ServerSocket which respectively define a platform independent server socket and connection (see implementing classes for platform specific implementation).

Implements a TCP connection framework, that is, a TCP client (TCPClient), a TCP server, which can open and close a server socket for a given address and port. The socket implementation is stored in TCPConnection, to ensure that the socket can be correctly closed. A TCP Listener is further implemented TCPListener, which will listen for incoming connections. Each received connection will then be handled in a separate thread by implementations of the interface TCPConnectionHandler.
